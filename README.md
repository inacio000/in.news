# Run project:
- Clone the repository
    ```bash
    git clone https://github.com/inacio000/in.news.git
    ```
- Install the dependencies
    ```bash
    npm install
    ```
- Run 
    ```bash
    npm run dev
    # or
    yarn dev
    ```

## Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
